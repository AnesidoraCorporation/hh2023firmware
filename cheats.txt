To solve the error code challenge, fill in the codes below using the shift register and boot / reset the badge. 
This will load in the code and one of the LEDs will turn off.

0b10000000

0b10110100

0b11101010

0b01101101

0b11011101

0b11010011

0b10011001

0b11100010

If you are halfway and want to reset this challenge and the guessing game, press the error button during boot.

If you filled in all the codes, the RGB led will blink blue. To load the brainfuck code quickly press error and then run. A menu will be available on the serial console
